**AutoDiag AI** is a next-generation Android application designed to bridge the gap between artificial intelligence and automotive mechanics. By harnessing the power of **Computer Vision** and **Generative AI**, it transforms any smartphone into a smart diagnostic scanner.

## ✨ Key Features

* 🔍 **Real-Time Object Detection:** Custom-trained **YOLOv8** model running on-device to instantly identify engine components.
* 🧠 **Smart Diagnostics (SCAN IA):** Integration with **Google Gemini 2.5 Flash** to analyze the physical state of parts and provide professional mechanical advice.
* 🗣️ **Voice Assistant (TTS):** Hands-free operation that reads diagnostic reports aloud—perfect for busy mechanics!
* ▶️ **Smart Repair Tutorials:** One-click integration with YouTube to instantly find targeted repair guides based on the exact faulty part.
* 🔦 **Built-in Flashlight:** Control the device's torch directly within the app for low-light garage environments.

* **CameraX API:** Highly optimized real-time image processing.
* **TensorFlow Lite Interpreter:** Offline edge computing for zero latency detection.
* **Google AI SDK:** Generative multimodal analysis via API.
